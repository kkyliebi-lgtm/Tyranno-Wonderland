Tyranno Wonderland
A website based on a creative design project by a visual designer Yingqi Bi that proposes a speculative language system for interaction between humans and Tyrannosaurus rex.

The project is set in a future where biotechnology may be able to reunite extinct ancient species. 

However, based on current information technology, traditional language is insufficient to address the vastly different cognitive and sensory structures of dinosaurs and humans.

To bridge this gap, the designers used semiotics and the history of ancient hieroglyphs (such as Chinese oracle bone script) to construct a visual system. 

This system's symbolic font transforms the skeletal structure and limb movements of dinosaurs into a shared graphic vocabulary, thereby promoting their coexistence in the city.

By blending critical design with historical typography, the project explores how visual communication can transcend the biological boundaries between species.
